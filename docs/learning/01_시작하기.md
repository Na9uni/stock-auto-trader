# 1. 시작하기 — 컴퓨터 켜고 나서 뭘 하면 되나?

## 아침에 컴퓨터 켜면

1. **영웅문4가 자동으로 열려요** (시작프로그램에 등록해놨어요)
2. 아이디 / 비밀번호 / 인증서 비밀번호 입력해서 로그인
3. 이 검은 창(Claude Code)을 열어요
4. **"오늘 시작할게"** 라고 입력하면 끝!

## Claude Code 여는 방법

- 바탕화면 아래 작업표시줄에서 **터미널** 또는 **명령 프롬프트** 찾기
- 또는 키보드에서 `Windows키 + R` → `cmd` 입력 → 엔터
- 그 다음 `claude` 입력 → 엔터

## 모르는 게 있으면?

그냥 **말로 물어보세요!** 예시:
- "이 프로그램이 뭘 하는 거야?"
- "지금 주식 시장 열려있어?"
- "백테스트가 뭐야?"
- "오늘 자동매매 결과 보여줘"

## 자주 쓰는 말

| 이렇게 말하면 | 이런 일이 일어나요 |
|---|---|
| "오늘 시작할게" | 시스템 상태 확인 + 준비 |
| "검색해줘" | 조건검색식으로 종목 찾기 |
| "백테스트 해줘" | 과거 데이터로 전략 테스트 |
| "자동매매 시작" | 프로그램이 알아서 매매 |
| "지금 상태 보여줘" | 보유 종목, 수익률 확인 |
| "오늘 끝!" | 작업 저장 + 정리 |
