# 3. 우리 프로그램이 하는 일

## 한마디로

**컴퓨터가 대신 주식을 사고 팔아주는 프로그램**이에요.

사람이 차트를 보고 "지금 사야겠다!" 하는 판단을 컴퓨터가 대신 해주는 거예요.

## 어떻게 동작하나?

```
1. 키움증권에서 실시간 주가 데이터를 가져와요 (30초마다)
2. 우리가 정한 조건(전략)에 맞는지 확인해요
3. 조건에 맞으면 자동으로 매수해요
4. 수익이 나면 자동으로 매도해요
5. 손해가 나면 자동으로 손절해요
6. 결과를 텔레그램으로 알려줘요
```

## 안전장치

프로그램에는 **안전장치**가 여러 개 있어요:

| 안전장치 | 내용 |
|---|---|
| 일일 손실 한도 | 하루 3만원 이상 잃으면 자동 중단 |
| 월간 손실 한도 | 한 달 10만원 이상 잃으면 자동 중단 |
| 연속 손절 한도 | 2번 연속 손절하면 자동 중단 |
| 매매 시간 제한 | 09:10 ~ 14:00 사이에만 매수 |
| 1회 매매 금액 | 25만원까지만 |
| 최대 보유 종목 | 동시에 2종목까지만 |

## 우리 전략 (변동성 돌파)

쉽게 말하면:
> "어제 주가가 많이 움직였으면, 오늘도 그만큼 움직일 거야.
> 오늘 시작가에서 어제만큼 올라가면 → 사자!"

그리고 **합산 거부권**이라는 보조 장치가 있어요:
> "다른 지표들이 '지금은 안 좋아'라고 하면 → 안 사!"

## 프로그램 구성

| 부분 | 역할 | 비유 |
|---|---|---|
| 키움 수집기 | 주가 데이터 가져오기 | **눈** (시장을 봐요) |
| 전략 | 사야 할지 판단 | **머리** (생각해요) |
| 자동매매 | 실제로 사고 팔기 | **손** (실행해요) |
| 텔레그램 | 결과 알려주기 | **입** (알려줘요) |
| 대시보드 | 상태 화면 보기 | **모니터** (보여줘요) |
