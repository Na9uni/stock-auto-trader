# 7. 차트 보는 법 — 기초 중의 기초

## 차트가 뭐야?

**주가의 변화를 그림으로 그린 것**이에요.

숫자만 보면 어려운데, 그림으로 보면 "올라가고 있네!", "내려가고 있네!" 한눈에 알 수 있어요.

## 캔들(봉)이란?

차트에 보이는 빨간/파란 막대기를 **캔들** 또는 **봉**이라고 해요.

```
    ┃  ← 윗꼬리 (하루 중 가장 높았던 가격)
   ┏┃┓
   ┃ ┃ ← 몸통 (시작~끝 가격 범위)
   ┗━┛
    ┃  ← 아랫꼬리 (하루 중 가장 낮았던 가격)
```

| 색깔 | 의미 |
|---|---|
| **빨간색** (양봉) | 올라간 날! 시작보다 끝이 높음 |
| **파란색** (음봉) | 내려간 날! 시작보다 끝이 낮음 |

## 이동평균선이란?

차트에 구불구불한 선이 보이시죠? 그게 **이동평균선**이에요.

| 선 | 의미 | 쉽게 말하면 |
|---|---|---|
| **5일선** | 최근 5일 평균 가격 | "이번 주 분위기" |
| **20일선** | 최근 20일 평균 가격 | "이번 달 분위기" |
| **60일선** | 최근 60일 평균 가격 | "최근 3개월 분위기" |
| **120일선** | 최근 120일 평균 가격 | "최근 반년 분위기" |

### 이동평균선으로 판단하는 법

- **주가가 이동평균선 위에 있으면** → 좋은 흐름! (상승 추세)
- **주가가 이동평균선 아래에 있으면** → 안 좋은 흐름 (하락 추세)
- **짧은 선이 긴 선을 위로 뚫으면** → 좋은 신호! (골든크로스)
- **짧은 선이 긴 선을 아래로 뚫으면** → 안 좋은 신호 (데드크로스)

## 거래량이란?

차트 아래에 있는 **막대그래프**가 거래량이에요.

- 거래량이 **많으면** = 관심 많은 종목 (사람들이 많이 사고 팔고 있음)
- 거래량이 **적으면** = 관심 없는 종목

### 중요한 포인트

> **주가가 오르면서 거래량도 같이 늘면** → 진짜 오르는 거!
> **주가가 오르는데 거래량이 줄면** → 힘이 빠지는 거... 조심!

## 영웅문4에서 차트 보는 법

1. 종목 검색란에 종목명 입력 (예: 삼성전자)
2. 차트가 자동으로 나와요
3. 상단에서 **일봉 / 주봉 / 월봉** 선택 가능
   - **일봉**: 하루 단위 (데이트레이딩에 주로 사용)
   - **주봉**: 일주일 단위 (큰 흐름 보기)
   - **월봉**: 한 달 단위 (장기 투자용)
