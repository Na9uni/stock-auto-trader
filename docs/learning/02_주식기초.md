# 2. 주식 기초 — 이것만 알면 돼요

## 주식이란?

회사의 일부를 사는 거예요. 삼성전자 주식 1주를 사면 삼성전자의 아주 작은 주인이 되는 거죠.

## 돈 버는 원리

**싸게 사서 비싸게 팔면 = 이익!**

예시:
- 삼성전자 1주를 **5만원**에 샀어요
- 나중에 **5만 5천원**이 됐어요
- 팔면 **5천원 이익!** (수수료 빼면 조금 줄어요)

## 꼭 알아야 할 용어

| 용어 | 쉬운 설명 |
|---|---|
| **매수** | 주식을 사는 것 |
| **매도** | 주식을 파는 것 |
| **호가** | 지금 사고 팔 수 있는 가격 목록 |
| **시가** | 오늘 장 열릴 때 첫 거래 가격 |
| **종가** | 오늘 장 마감 때 마지막 거래 가격 |
| **거래량** | 오늘 이 주식이 몇 번 거래됐는지 |
| **손절** | 손해를 감수하고 파는 것 (더 큰 손해 막기 위해) |
| **익절** | 이익이 날 때 파는 것 |
| **수수료** | 주식 사고 팔 때 증권사에 내는 비용 |

## 장 운영 시간

| 시간 | 상태 |
|---|---|
| 09:00 ~ 15:30 | **정규장** (이때 매매 가능) |
| 15:30 ~ 16:00 | 장 종료 |
| 토/일/공휴일 | **쉬는 날** (매매 불가) |

## 데이트레이딩이란?

- **오늘 사고 오늘 파는** 매매 방식
- 아침에 사서 오후에 팔아요
- 1~3% 이익을 목표로 해요
- **절대로** 밤새 들고 있으면 안 돼요 (위험!)

## 가장 중요한 3가지 규칙

1. **욕심 금지** — 2~3% 먹으면 바로 팔기
2. **손절 필수** — 떨어지면 미련 없이 팔기
3. **소액으로** — 처음엔 적은 돈으로 연습하기
