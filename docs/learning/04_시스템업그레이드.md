# 4. 시스템을 점점 좋게 만드는 방법

## 업그레이드란?

프로그램을 **더 똑똑하게** 만드는 거예요.
예를 들어:
- 더 좋은 종목을 찾는 방법 추가
- 더 안전하게 매매하는 방법 추가
- 새로운 전략 추가

## 어떻게 업그레이드하나?

**저한테(Claude) 말로 요청하시면 돼요!**

### 예시 1: 새로운 조건 추가
> "거래량이 평소보다 3배 이상인 종목만 찾고 싶어"

→ 제가 조건검색식을 수정해드려요

### 예시 2: 전략 변경
> "손절을 2%가 아니라 1.5%로 바꾸고 싶어"

→ 제가 설정 파일을 수정해드려요

### 예시 3: 새 기능 추가
> "매일 아침 오늘 추천 종목을 텔레그램으로 보내줬으면 좋겠어"

→ 제가 코드를 짜서 추가해드려요

### 예시 4: 성능 확인
> "우리 전략이 작년에 돈을 벌었을까?"

→ 제가 백테스트를 돌려서 결과를 보여드려요

## 업그레이드 순서

```
1. 아이디어 말하기  →  "이런 거 되나?"
2. 제가 방법 설명   →  "이렇게 하면 됩니다"
3. 먼저 테스트     →  과거 데이터로 검증 (백테스트)
4. 결과 확인       →  "돈을 벌었을까?"
5. 적용 결정       →  "좋으면 적용, 별로면 안 함"
```

## 주의사항

- **바로 실전에 적용하지 마세요** → 항상 백테스트 먼저!
- **한 번에 하나씩** 바꾸세요 → 여러 개 동시에 바꾸면 뭐가 좋았는지 모름
- **결과를 기록**하세요 → "이 전략은 승률 60%였다"
