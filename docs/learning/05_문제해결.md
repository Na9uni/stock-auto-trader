# 5. 문제가 생겼을 때

## 당황하지 마세요!

뭔가 안 되면 저한테 **그대로 말해주세요**. 예시:

- "에러가 떴어" → 화면 캡처해서 보여주세요
- "영웅문이 안 열려" → 어떤 메시지가 나오는지 알려주세요
- "자동매매가 안 돼" → 제가 로그를 확인해드려요

## 자주 생기는 문제

### "영웅문4 로그인이 안 돼요"
→ 인증서 만료됐을 수 있어요. "인증서 갱신해줘"라고 말씀해주세요.

### "프로그램 실행했는데 에러가 나요"
→ 화면을 캡처해서 보여주세요. 제가 바로 고쳐드려요.

### "텔레그램 알림이 안 와요"
→ "텔레그램 확인해줘"라고 말씀해주세요.

### "아무것도 모르겠어요"
→ 전혀 괜찮아요! "처음부터 다시 알려줘"라고 하시면 됩니다.

## 절대 하면 안 되는 것

1. **모르는 파일 삭제하지 마세요** (중요한 파일일 수 있어요)
2. **설정 파일(.env)을 직접 수정하지 마세요** (저한테 말씀해주세요)
3. **.git 폴더 건드리지 마세요** (코드 이력이 다 들어있어요)
