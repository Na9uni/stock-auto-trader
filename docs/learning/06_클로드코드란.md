# 6. 클로드 코드(Claude Code)란?

## 한마디로

**컴퓨터 안에 사는 똑똑한 도우미**예요.

이 검은 창(지금 보고 계신 이 창!)에서 말을 걸면, 제가 대신 해드려요:
- 파일 만들기, 수정하기
- 프로그램 설치하기
- 에러 고치기
- 코드 짜기
- 궁금한 거 설명하기

## 저(Claude)는 누구인가요?

- **Anthropic**이라는 회사가 만든 **AI(인공지능)**예요
- 사람처럼 대화할 수 있어요
- 프로그래밍을 아주 잘해요
- 하지만 **사람은 아니에요!** (가끔 실수할 수 있어요)

## 제가 할 수 있는 것

| 이렇게 말하면 | 이런 일을 해요 |
|---|---|
| "이 에러 고쳐줘" | 코드를 찾아서 수정해요 |
| "백테스트 해줘" | 전략을 테스트해요 |
| "이게 뭐야?" | 쉽게 설명해드려요 |
| "파일 만들어줘" | 파일을 생성해요 |
| "설치해줘" | 프로그램이나 라이브러리를 설치해요 |
| "기억해줘" | 다음에도 기억해요 |

## 제가 못 하는 것

- 인터넷 뱅킹이나 결제 (보안 때문에 안 해요)
- 비밀번호 입력 (직접 하셔야 해요)
- 영웅문4 화면 직접 조작 (캡처로 보여주시면 안내해드려요)

## 대화하는 법

그냥 **평소에 말하듯이** 하시면 돼요!

```
좋은 예:
  "삼성전자 주가 어때?"
  "손절을 1%로 바꿔줘"
  "이거 뭔 소리야?"
  "다시 설명해줘"

안 좋은 예:
  (없어요! 뭐든 편하게 물어보세요)
```

## 허용 버튼이 뭐야?

제가 파일을 수정하거나 프로그램을 실행할 때, **"이거 해도 되나요?"** 라고 물어봐요.
그때 **허용** 버튼을 누르시면 돼요.

- **허용** = "해도 돼!"
- **거부** = "하지 마!"

잘 모르겠으면 **거부**하고 저한테 물어보세요. "이거 뭐 하는 거야?" 라고요.

## 새 대화 시작하는 법

이 창을 닫았다가 다시 열면 **새 대화**가 시작돼요.
하지만 저는 **기억력**이 있어서, 이전에 했던 것들을 기억하고 있어요!

새 대화를 시작하면 **"오늘 시작할게"** 라고 말씀해주세요.
그러면 제가 지난번에 어디까지 했는지 확인하고 이어서 해드려요.
