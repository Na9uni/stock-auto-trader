# Trading Review Skill (도메인 기반 자동 리뷰)

전략 코드를 **트레이딩 도메인 지식** 관점에서 리뷰한다.
코드 문법이나 구조가 아니라, **이 로직이 실제 시장에서 작동하는가**를 검증한다.

## 리뷰 체크리스트

### 1. 가격-거래량 관계
- [ ] 거래량 급증 + 가격 하락을 매수 신호로 오해석하고 있지 않은가? (= 분배)
- [ ] 거래량 없는 돌파를 유효한 돌파로 취급하고 있지 않은가?
- [ ] 시가 갭 처리가 현실적인가? 갭에서 체결 불가능한 가격으로 진입하지 않는가?

### 2. 지표 사용 적정성
- [ ] RSI 기간 14 표준. 5 이하 과민, 30 이상 둔감. 극단값 설정 시 근거?
- [ ] MACD 12/26/9 표준. 변경 시 의미 이해?
- [ ] 여러 지표 조합 시: 같은 데이터를 다른 방식으로 보는 중복 지표는 아닌가?

### 3. 시장 미시구조
- [ ] 슬리피지/스프레드 반영이 현실적인가?
- [ ] "이 가격에 무조건 체결" 가정이 있는가? 유동성 부족 종목은?
- [ ] 동시호가 시간대 처리?

### 4. 백테스트 함정
- [ ] **Lookahead bias**: 당일 고가/저가를 장중에 사용?
- [ ] **Survivorship bias**: 상장폐지 종목 누락?
- [ ] **과적합**: walk-forward 검증? in/out-sample 분리?
- [ ] **거래 횟수**: 30회 미만이면 통계적 유의성 부족

### 5. 레짐 적합성
- [ ] 추세추종이 횡보장에서 어떻게 작동하는가?
- [ ] 역추세 전략이 강한 추세장에서 역방향 진입하지 않는가?
- [ ] 위기 모드 전환 타이밍이 너무 늦지 않은가?

### 6. 리스크 관리
- [ ] 단일 종목 비중 제한?
- [ ] 같은 섹터 동시 보유 시 실질 분산?
- [ ] 연속 손실 시 포지션 축소 로직?
- [ ] MAX_DAILY_LOSS, MAX_MONTHLY_LOSS 현실적?

### 7. 한국 시장 특수성
- [ ] 외국인/기관 수급 활용 여부?
- [ ] 선물/옵션 만기일 변동성 처리?
- [ ] 대형주 vs 중소형주 특성 구분?

## 리뷰 보고 형식

```
## 도메인 리뷰: <변경 파일>

### 시장 원리 부합성
- [적합/주의/부적합] 근거: ...

### 발견된 문제
1. [높음/중간/낮음] 설명 + 개선 제안

### 개선 제안 (묻지 않아도 제시)
1. ...

### 판정: PASS / WARN / BLOCK
```
