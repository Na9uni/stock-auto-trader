# 세션 종료 스킬

## Steps

1. 오늘 한 작업 요약 (3줄 이내)
2. 변경된 파일이 있으면 git commit 제안
3. 다음에 할 일 정리
4. 메모리 업데이트 (새로 배운 것, 진행 상황)

## 아빠인 경우

- 쉬운 말로 요약
- "내일은 이런 거 해볼 수 있어요" 제안
- 칭찬 한마디 ("오늘 많이 배우셨어요!")

## 아들인 경우

- 기술적 요약 (변경 파일, 커밋 내역)
- 남은 과제 목록
- PR 필요 여부 확인
