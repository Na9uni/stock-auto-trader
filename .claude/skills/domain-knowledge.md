# Trading Domain Knowledge

전략 코드 작업 시 반드시 참조해야 할 트레이딩 도메인 지식.
**이 지식을 코드 리뷰, 전략 설계, 백테스트 해석에 적극 활용하라.**

## 가격-거래량 핵심 원리

### 정상 관계 (Confirmation)
- 가격 상승 + 거래량 증가 = 건강한 상승 추세 (매수세 유입)
- 가격 하락 + 거래량 증가 = 건강한 하락 추세 (매도세 유입)

### 다이버전스 (경고 신호)
- 가격 상승 + 거래량 감소 = 추세 약화, 반전 가능 (매수세 소진)
- 가격 하락 + 거래량 감소 = 매도세 소진, 바닥 근접 가능
- 가격 횡보 + 거래량 급증 = 큰 움직임 임박 (방향 미정)

### 세력/수급 패턴
- 거래량 급증 + 가격 소폭 상승 = 매집(accumulation)
- 거래량 급증 + 가격 소폭 하락/횡보 = 분배(distribution)
- **이것을 거꾸로 해석하면 큰 손실로 이어짐**

## 기술적 분석 핵심

### 지지/저항
- 이전 고점 = 저항, 이전 저점 = 지지
- 뚫린 저항 → 새로운 지지로 전환
- 여러 번 테스트된 레벨이 더 강함

### 이동평균 해석
- 골든크로스/데드크로스: 후행 지표 — 이미 상당히 움직인 후 발생
- MA 수렴 → 발산: 새로운 추세 시작 신호

### 변동성 돌파 (Volatility Breakout)
- 전일 변동폭의 K배를 당일 시가에 더한 가격 돌파 시 매수
- K값: 0.4~0.6이 한국 시장에서 일반적
- 약점: 횡보장 거짓 돌파 빈번, 갭 상승 시 진입 불가

### 추세추종 vs 역추세
- 추세추종: 승률 낮지만 이익/손실 비율(RR) 높음
- 역추세(평균회귀): 승률 높지만 RR 낮음
- 결합하면 서로 약점 보완 가능
- 한국: 대형주는 추세 성향, 코스닥 중소형은 평균회귀/테마 성향 상대적으로 강함

## 백테스트 해석 가이드

### 핵심 지표 해석
- **Sharpe**: 1.0+ 양호, 0.5 미만 재검토. 한국 단타 0.5~1.0도 현실적
- **PF**: 총이익/총손실. 1.5+ 양호, 1.2 미만 위험
- **MDD**: 20%+ 이면 실전 심리적 운용 어려움
- **승률**: 단독 무의미. 승률 30% + RR 3:1이면 수익

### 흔한 함정
- **Lookahead bias**: 당일 고가/저가를 장중에 사용 불가
- **Survivorship bias**: 상장폐지 종목 누락
- **과적합**: walk-forward, in/out-sample 분리로 검증
- **거래 횟수**: 30회 미만은 통계적 유의성 부족

## 한국 시장 실전 지식

### 수급 구조
- 외국인/기관 수급이 시장 방향 결정 비중 큼
- 개인 순매수 = 역지표인 경우 다수
- 선물/옵션 만기(매월 둘째 주 목요일) 전후 변동성 왜곡

### 시간대별 특성
- 09:00~09:30: 변동성 최대, 거짓 신호 많음
- 09:30~11:00: 추세 형성, 주요 매매 시간대
- 11:00~14:00: 횡보 구간
- 14:00~15:30: 마감 수급 정리

### 비용 현실 (⚠️ 아래 수치는 수시 변동 — 사용자에게 확인 후 적용)
- 키움 수수료: 약 0.015% (HTS/API 기준, 이벤트에 따라 변동)
- 매도세(증권거래세): 약 0.18% (정책에 따라 변동, 최신 세율 확인 필요)
- 슬리피지: 소형주 0.3~0.5%, 대형주 0.05~0.1%
- 가격 제한폭: 상한/하한 30% (최신 규정 확인 필요)

## 전략 개선 시 점검 순서

성과가 안 나올 때 — 진입보다 **청산**이 문제인 경우가 대부분:
1. 비용 모델이 현실적인가?
2. **청산 로직**에 문제가 없는가? (대부분의 문제는 여기)
3. 포지션 사이징이 적절한가?
4. 레짐 필터가 작동하는가?
5. 마지막으로 진입 로직 점검

### 필터 추가 우선순위
1. 거래량 필터 — 가장 효과적
2. 레짐 필터 — 두 번째
3. 시간대 필터 — 세 번째
4. 섹터/테마 필터 — 상황별
